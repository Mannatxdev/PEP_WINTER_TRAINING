## CREATE PROJECT ##
1. django-admin startproject portfolio
2. cd portfolio
3. python manage.py runserver

## Create App (Your Website) ##
1. python manage.py startapp home


## Installing Django-tailwind ##
1.  uv pip install django-tailwind
2.  uv pip install django-tailwind[#cookiecutter,honcho,reload]
3.  cd portfolio
4.  python manage.py tailwind init


## Installing NODE ##
1. node -v
2. npm -v
3. pip install django-browser-reload
4. npm init -y
5. npm install tailwindcss postcss autoprefixer
